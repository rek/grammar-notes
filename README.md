# grammar-notes [![Build Status](https://travis-ci.org/rek/grammar-notes.svg?branch=master)](https://travis-ci.org/rek/grammar-notes)

To help with note taking when writing a grammar.

## To run in dev:

```
    $ npm install
    $ gulp
```

## To make a build:

```
    $ npm install
    $ gulp build
```

## To test:

```
    $ npm install
    $ gulp test
```
